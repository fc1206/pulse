## The actionability bar (what earns a digest item)

A finding is digest-worthy only if it would change a decision in the next 2–8 weeks. Most findings are registry-worthy but NOT digest-worthy — that's correct. A digest item must tie to a pillar or a standing read above and end in a concrete, ownable action (a battlecard, a positioning line, a deck slide, a build/decision call). If it doesn't clear that bar, the honest output is `NO ACTIONABLE SIGNAL`.
